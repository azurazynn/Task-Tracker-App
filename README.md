# 📝 My Task Tracker App

A simple and effective mobile application designed to help you organize your daily tasks with a clean, modern interface.

---

## 📋 Key Features

* **Add Tasks:** Quickly create new tasks by typing in the box and clicking "Add."
* **Track Progress:** See exactly how many tasks you have left with the live "Items Remaining" counter.
* **Mark as Done:** Click the checkbox to cross off finished tasks and stay organized.
* **Delete Tasks:** Easily remove any task from your list using the delete icon.
* **Modern Design:** Enjoy a clean and simple look with a smooth blue-gradient background.
